# LyricDumper

HOW TO USE LYRICS DUMPER
1. you will need a file called "data.txt" and "song.wav"

w:2100:n:The Quick brown fox jumps over the lazy dog;
the "w" is the colour, w = white (LOOK AT THE LIST BELOW TO SEE THE COLOUR CODE) 2100 is the delay in ms. and the n will make it so it will type out on a new line, put a space instead if you want the text to be on the same line. the final bit is the text printed out, ALL LINES MUST END WITH A SEMI COLON EXCEPT THE LAST ONE.

w - white
r - red
b - blue
c - cyan
g - green
y - yellow
m - magenta
dm - dark magenta
gr - gray
bl - black
db - dark blue
dc - dark cyan
dr - dark red
dy - dark yellow
dg - dark green
dgr - dark grey
